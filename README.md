易信 / 微信 公众平台_查快递
======

使用方法,编辑以下内容发送至公众平台即可:   
如（查询EMS单号为```1034616494006```的快递
```ems 1034616494006``` 

只做一个快递公司，后期会添加快递100API没有的一些快递。   
比如易客满(美国亚马逊海淘至中国使用的快递)
